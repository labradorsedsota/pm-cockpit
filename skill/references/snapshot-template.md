# 上下文快照模板

跨 session 任务启动前，或阶段性保存时，写入 memory 文件。

## 模板

```markdown
# {项目代号} 上下文快照 — {日期}

## 当前进度
- 阶段：{需求/开发/测试/修复/回归/验收}
- 最新完成：{最后完成的事项}
- 进行中：{当前正在做的事项}

## 下一步计划
1. {下一步 1}
2. {下一步 2}

## 已知约束
- {约束 1}
- {约束 2}

## 关键文件位置
- PRD：{路径}
- Test Case：{路径}
- Test Report：{路径}
- Repo：{URL}

## 待确认事项
- {需要确认的问题}
```

## 使用时机

- 预计 session 可能中断时（长时间任务、等待他人回复）
- 每个大阶段完成时
- 遇到需要暂停的决策点时
